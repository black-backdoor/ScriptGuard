Things ScriptGuard can and can't do

ScriptGuard can:

  • Scan userscripts for malicious code
  • Warn you about malicious code
  • Be used as a website
  • Be used as a web app
  • Be used for free
  • Be used as open source

ScriptGuard can't:

  • It cannot activly protect you from malicious code
  • It cannot detect all malicious code
